Explicacion de C++

Codigo

Para comenzar se pondria el include y despues se abriria el main y comenzaria a codificarr dentro del main

despues dentro del main se comenzaria por colocar un printf y dentro del printf se colocaria el enunciado "dame una cadena de caracteres"

despues de eso se colocaria un scanf que guardaria la cadena de caracteres en una variable que llamaremos caracteres

una ves se tenga la variable guardada se usara un if para comparar que sea un string o cadena de caracteres

si es asi le dejaremos seguir y con una funcion que contara el numero de palabras que tiene la cadena de caracteres veremos si es string o no

y por ultimo le imprimiremos el numero de la cadena de caracteres si es string

de lo contrario se usara un else el cual le imprimira que esa no es una cadena de caracteres

esto se repetira 3 veces mas, una ves se gaste sus 3 oportunidades se cerrara el programa
